About

Shutterfly provides their customers with the ability to create custom products using their own photos (photo albums, mugs, stationary, etc.).

When their users upload vacation photos, they are interested in knowing the location in which the photos were taken, in order to provide custom designs for famous sites. Not all of the photos come with geolocation information, so they need the ability to recognize famous landmarks from pixels alone.

This project provides an exploration of a deep learning solution to this problem.


Goals

Gain knowledge in Deep Learning

Learn about the subject, focusing on image classification

Find existing dataset

Search the web for a dataset that will resemble the images shutterfly users upload

Explore CNN based solution

convolutional neural networks (CNN) are widely used for image recognition, we tested their performance on our problem

Conclude and share findings

Methodology

Google Landmarks Dataset from kaggle

Pytorch Framework

AWS instance equipped with an Nvidia GPU

In order to better understand the specific challenges of this problem, we started from smaller problems and gradually moved on to bigger ones. At each stage we created custom datasets containing different amounts of landmarks, starting from 1 landmark and ending up with 1K landmarks.

We trained models using existing CNN architectures pretrained on the ImageNet dataset, at this point classifying single images.

After gaining insights from the smaller problems we focused on the 1K landmarks dataset and tested the current solution limitations.

Shutterfly users usually uploads several images from the same vacation, we tried to take advantage of this and classify sequences of images instead of one image at a time.

We combined our trained CNN with LSTM (recurrent neural network capable of learning long-term dependencies) to create a new model. The input to this model is a sequence of images that are known to be from the same class, and the output is the predicted class.

Achievements

  • We found that the landmark recognition problem is approachable using deep learning and CNN in particular
  • We provided insights that can be used in classifying real-life images of Shutterfly's users
  • Our final LSTM approach boosted the original CNN accuracy by 10%,
    and proved to be more accurate than a simple majority vote baseline
